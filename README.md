# BrainTumorPredictViewer

## Overview

- TensorFlow를 사용한 뇌종양 예측 프로그램의 시각화

## Role

- 정재우: 기존 만들어 둔 뇌종양 예측 코드 및 테스트 이미지 준비
- 김종필: python 파일의 실행 및 테스트 이미지 및 실행 결과 폴더에 대한 Viewer 준비
